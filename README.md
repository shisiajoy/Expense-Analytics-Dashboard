# 💰 Expense Dashboard

An interactive personal finance analytics dashboard built using Streamlit.

## 🚀 Features
- Interactive filtering (date & category)
- Real-time spending metrics
- Visual analytics (bar & line charts)
- Insights generation
- Dynamic synthetic dataset

## 🧠 Tech Stack
- Streamlit
- Pandas
- Plotly
- Python

## 📊 What it shows
- Total spending trends
- Category-wise breakdown
- Spending patterns over time
- Highest expense insights

## ▶️ How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
